# Student_Management_System

## Tools Used

* Python 
* Tkinter Module
* MySQL


## Description
We can perform operations like add student details, update student details, delete student details, and search
student details by their roll no, name or contact.


## IDE Used

Visual Studio Code


## Localhost Used

XAAMP Localhost
